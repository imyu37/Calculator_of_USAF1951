# USAF 1951 空间频率计算器

这是一个基于Python和FreeSimpleGUI开发的USAF 1951分辨率测试靶标准计算器。该工具可以帮助用户快速计算USAF 1951标准中的空间频率和对应的线宽。

## 功能特点

- 计算USAF 1951标准中的空间频率（单位：lp/mm）
- 支持多种单位显示（毫米、微米、纳米）
- 实时显示线宽信息
- 提供完整的USAF 1951参考表
- 清晰的图形用户界面

## 计算公式

- **空间频率**: f = 2^(group + (element-1)/6) [lp/mm]
- **线宽**: line_width = 1/(2*f) [mm]

## 界面说明

### 输入参数
- **组 (Group)**: 范围从-2到9，代表不同的分辨率级别
- **元素 (Element)**: 范围从1到6，在每组中有6个不同的元素
- **单位选择**: 可选择毫米(mm)、微米(μm)或纳米(nm)

### 输出结果
- 空间频率值
- 不同单位下的线宽
- 模式描述
- 物理意义说明

## 安装依赖

```bash
pip install FreeSimpleGUI
```

## 运行程序

```bash
python USAF_1951_空间频率计算器.py
```

## 使用方法

1. 从下拉菜单中选择组号和元素号
2. 选择需要的单位
3. 点击“计算”按钮查看结果
4. 点击“显示全部模式”查看完整的USAF 1951参考表
5. 点击“清除”清空结果区域

## 应用场景

- 光学系统分辨率测试
- 相机镜头性能评估
- 显微镜物镜校准
- 半导体检测设备校准
- 精密光学元件检测

## 组别说明

- Group -2 to 0: 大型光学系统, 投影仪
- Group 1 to 3: 相机镜头, 望远镜
- Group 4 to 5: 显微镜, 精密仪器
- Group 6 to 7: 半导体检测, 科研设备
